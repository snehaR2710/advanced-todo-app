# The Todo App

A fully responsive React todo application built with Vite and Tailwind CSS.


## 🔗 Live Demo
[View Live App](https://advanced-todo-app-rose.vercel.app)

## ✨ Features
- Add, edit, delete tasks
- Mark tasks as done / undo with strikethrough
- Filter by All / Pending / Completed
- localStorage persistence — tasks saved on refresh
- Responsive on all devices — mobile, tablet, desktop

## 🛠️ Tech Stack
- React 18
- Tailwind CSS
- Vite
- localStorage API
